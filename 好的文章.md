## 符号运算
```python
from sympy import *

a, b = symbols('a b')
v = Matrix([a, b])
print(v @ v.T)
```

求逆矩阵
```python
from sympy import *

a, b, c, d = symbols('a b c d')
M = Matrix([[a, b],
            [c, d]])
print(M)
print(M.T)
print(M.inv())
```
